# LEDerZaumzeug
LED Generator WS2801 WS2812b etc.

Geschrieben in C# für .Net core (auf auch Linux)
Mit Konfigurationsdateien, später mit GUI
Ein Generator schöner Grafikanimationen für LED-Matrixen.
Plug-In-Architektur
verschiedene Outputs etc.
